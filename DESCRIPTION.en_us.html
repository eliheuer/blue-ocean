Blue Ocean is a typeface originally forked from VTF Bluu Next by Jean-Baptiste Morizot, now with an expanded weight range redrawn in quadratic curves by Eli Heuer. The quadratic curve drawing style gives the font a phototypesetting-like quality, and the outline structure allows for interpolation between a wide range of forms.

The Blue Ocean project is led by Eli Heuer, a type designer from Seattle. Pull requests and new issues are welcome. To contribute, see: github.com/eliheuer/blue-ocean
