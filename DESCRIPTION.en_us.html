Blue Ocean is a fork of the typeface VTF Bluu Next by Jean-Baptiste Morizot, now with an expanded weight range redrawn in quadratic curves by Eli Heuer. The quadratic curve drawing style gives the font a smooth phototypesetting-like quality.

The Blue Ocean project is led by Eli Heuer, a type designer based in Seattle, WA. Pull requests and new issues are welcome. To contribute to the development of this typeface, visit: https://github.com/eliheuer/blue-ocean
